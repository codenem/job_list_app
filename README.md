## JOB LIST APP

### Requirements

1. Install node.js
2. Run:
```
npm install -g bower
```

### Local Installation

1. Run:
```
cd path/to/root/folder/
```
```
bower install
```
1. Open index.html with Chrome (I didn't test on FF nor IE).
